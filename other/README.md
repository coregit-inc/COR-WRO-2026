# Other Files

This folder contains additional documentation and resources.

## Contents

- `test_results.xlsx` - Spreadsheet with test run data and performance metrics
- `calibration_notes.txt` - Sensor calibration values and procedures
- `engineering_journal.pdf` - Full engineering journal (paper version)

## Testing Protocol

1. Place robot in start zone
2. Power on EV3 brick
3. Press center button to load program
4. Press right arrow to start
5. Record: lap times, completion status, wall contacts, obstacle avoidance results
6. Log data in test_results.xlsx
