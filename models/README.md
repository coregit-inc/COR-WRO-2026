# 3D Models and Assembly Instructions

This folder contains files related to the physical design of the vehicle.

## Files

- `FE-Robot.io` - 3D model file for Studio 2.0 (BrickLink)
- `Robot_Instruction.pdf` - Step-by-step assembly instructions

## Software Required

- **Studio 2.0** by BrickLink (free) - for viewing and editing the 3D model
  - Download: https://www.bricklink.com/v3/studio/download.page

## Parts List

The vehicle is built primarily from:
- LEGO EV3 MINDSTORMS Education Core Set (45544)
- LEGO EV3 MINDSTORMS Education Expansion Set (45560)
- Additional LEGO Technic beams and connectors
- Pixy2 Camera with LEGO mounting bracket
